Aim of the Experiment:

Interfacing Light Dependent Resistor (LDR) and LED, displaying automatic night lamp

Procedure:

  1. Connect the LDR to A0 using a voltage divider:
    - One leg of the LDR goes to 5V.
    - Second leg to A0 and one end of 10kŒ© resistor.
    - Other end of resistor to GND.
  2. Connect the LED to digital pin 9 through a 220Œ© resistor:
    - Anode (+) of LED ‚Üí Pin 9 via 220Œ© resistor
    - Cathode (‚Äì) of LED ‚Üí GND
  3. Write and upload the Arduino code:
    - Use analogRead() for LDR and digitalWrite() for LED control.
    - Upload the code using Arduino IDE.
  4. Open the Serial Monitor:
    - Set baud rate to 9600 to observe real-time LDR values.
  5. Test in different lighting conditions:
    - Cover the LDR: LED should turn ON (dark condition).
    - Shine a light on LDR: LED should turn OFF (bright condition).
  6. Verify output and adjust threshold if needed:
    - Observe LED and serial monitor values.
    - Adjust threshold in the code to suit room lighting.

Code:

int ldrPin = A0;      // LDR connected to analog pin A0
int ledPin = 9;       // LED connected to digital pin 9
int ldrValue = 0;     
int threshold = 500;  // Adjust this value based on your room light

void setup() {
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600); // For debugging
}

void loop() {
  ldrValue = analogRead(ldrPin);
  Serial.println(ldrValue); // See the LDR value on Serial Monitor

  if (ldrValue < threshold) {
    digitalWrite(ledPin, HIGH);  // It's dark, turn ON LED
  } else {
    digitalWrite(ledPin, LOW);   // It's bright, turn OFF LED
  }

  delay(500);
}
