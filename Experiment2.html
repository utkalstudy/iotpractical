Experiment-2

Aim of the Experiment:

Interfacing Button and LED ‚Äì LED blinking when button is pressed.

Hardware Required:

  • Arduino UNO board
  • LED (any colour)
  • 220-ohm resistor
  • Breadboard
  • Jumper wires
  • USB cable for Arduino
  • Push Button

Software Required:

  • Arduino IDE

Procedure:

  1. Connect the anode of the LED to Arduino digital pin 9 via a 220Œ© resistor
  2. Connect the cathode of the LED to the ground (GND).
  3. Open Arduino IDE and create a new sketch.
  4. Connect one terminal of the push button to 5V.
  5. Connect the other terminal of the push button to Arduino digital pin 2

Code:

const int buttonPin = 2; 
const int ledPin = 9; 
void setup() { 
    pinMode(buttonPin, INPUT); 
    pinMode(ledPin, OUTPUT); 
} 
void loop() { 
    if (digitalRead(buttonPin) == HIGH) { 
        digitalWrite(ledPin, HIGH); 
        delay(500); 
        digitalWrite(ledPin, LOW); 
        delay(500); 
    } else { 
        digitalWrite(ledPin, LOW); 
    } 
}

Output

  • When we click push button, the light will glow.
