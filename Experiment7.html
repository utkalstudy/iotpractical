Aim of the Experiment:

Interfacing Bluetooth Module (HC-05) ‚Äì Receiving data from mobile phone on Arduino and display on LCD

Hardware Required:

  • Arduino Uno ‚Äì 1
  • HC-05 Bluetooth Module ‚Äì 1
  • 16x2 LCD Display ‚Äì 1
  • 10kŒ© Potentiometer ‚Äì 1
  • Jumper Wires ‚Äì As needed
  • Breadboard ‚Äì 1
  • USB Cable for Arduino ‚Äì 1
  • Mobile Phone with Bluetooth Terminal App

Software Required:

  • Arduino IDE
  • LiquidCrystal Library

Procedure:

  1. Connect the LCD:
    - RS ‚Üí pin 12, EN ‚Üí pin 11
    - D4 ‚Üí pin 5, D5 ‚Üí pin 4, D6 ‚Üí pin 3, D7 ‚Üí pin 2
    - Potentiometer middle pin to V0 (contrast), sides to 5V and GND
  2. Connect the HC-05 Bluetooth Module:
    - VCC ‚Üí 5V, GND ‚Üí GND
    - TX ‚Üí RX of Arduino (pin 0), RX ‚Üí TX of Arduino (pin 1)
    - (Use voltage divider for RX if needed)
  3. Upload the code:
    - Disconnect Bluetooth while uploading code
    - Reconnect after upload
  4. Open Bluetooth Terminal App:
    - Pair with HC-05 using default PIN (1234 or 0000)
    - Send any text via terminal
  5. Observe Output:
    - LCD will display the received text from mobile

Code:

#include <LiquidCrystal.h>

LiquidCrystal lcd(12, 11, 5, 4, 3, 2);

String incomingData = "";

void setup() {
  lcd.begin(16, 2);
  lcd.print("Waiting for Data");
  Serial.begin(9600); // HC-05 default baud rate
}

void loop() {
  if (Serial.available()) {
    char received = Serial.read();
    if (received == '\n') {
      lcd.clear();
      lcd.setCursor(0, 0);
      lcd.print("Received:");
      lcd.setCursor(0, 1);
      lcd.print(incomingData);
      incomingData = "";
    } else {
      incomingData += received;
    }
  }
}

Output

  • Data sent from the mobile via Bluetooth appears on the LCD.
  • Example: Sending "Hello" shows "Received: Hello" on LCD.
