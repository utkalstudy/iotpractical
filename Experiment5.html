Interfacing Liquid Crystal Display (LCD) ‚Äì Display data generated by sensor on LCD

Software Required:

  • Arduino IDE
  • LiquidCrystal Library
  • DHT Sensor Library

Procedure:

  1. Install required libraries:
    - In Arduino IDE, go to Sketch > Include Library > Manage Libraries
    - Install "LiquidCrystal" and "DHT sensor library"
  2. Make the connections:
    - LCD RS to digital pin 12
    - LCD EN to digital pin 11
    - LCD D4 to digital pin 5
    - LCD D5 to digital pin 4
    - LCD D6 to digital pin 3
    - LCD D7 to digital pin 2
    - Potentiometer center pin to LCD V0 (contrast), sides to 5V and GND
    - DHT11 VCC to 5V, GND to GND, DATA to pin 7
  3. Upload the code:
    - Use the code below to read temperature and humidity and display on LCD
  4. Observe Output:
    - LCD should display "Temp: XX¬∞C" and "Humidity: XX%"

Code:

#include <LiquidCrystal.h>
#include <DHT.h>

#define DHTPIN 7
#define DHTTYPE DHT11

DHT dht(DHTPIN, DHTTYPE);

// Initialize the LCD (rs, en, d4, d5, d6, d7)
LiquidCrystal lcd(12, 11, 5, 4, 3, 2);

void setup() {
  lcd.begin(16, 2); // 16 columns, 2 rows
  dht.begin();
  lcd.print("Initializing...");
  delay(2000);
}

void loop() {
  float temp = dht.readTemperature();
  float humid = dht.readHumidity();

  if (isnan(temp) || isnan(humid)) {
    lcd.clear();
    lcd.print("Sensor Error");
    return;
  }

  lcd.clear();
  lcd.setCursor(0, 0);
  lcd.print("Temp: ");
  lcd.print(temp);
  lcd.print("C");

  lcd.setCursor(0, 1);
  lcd.print("Humidity: ");
  lcd.print(humid);
  lcd.print("%");

  delay(2000); // Update every 2 seconds
}

Output

  • The LCD displays real-time temperature and humidity data from the sensor.
