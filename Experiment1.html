Experiment-1

Aim of the Experiment:

Interfacing Light Emitting Diode (LED) - Blinking LED

Hardware Required:

  • Arduino UNO board
  • LED (any colour)
  • 220-ohm resistor
  • Breadboard
  • Jumper wires
  • USB cable for Arduino

Software Required:

  • Arduino IDE

Procedure:

  1. Install Arduino IDE on your computer.
  2. Connect the Arduino Uno to the PC via a USB cable.
  3. Open Arduino IDE and create a new sketch.
  4. Copy and paste the below code into the Arduino IDE.
  5. Select the correct Board: Go to Tools > Board > Arduino Uno.
  6. Select the correct Port: Go to Tools > Port > COMx (where Arduino is connected).
  7. Click on Upload to upload the code.

Code:

int ledpin = 13;

void setup() {
  pinMode(ledpin, OUTPUT);
}

void loop() {
  digitalWrite(ledpin, HIGH);
  delay(1000);
  digitalWrite(ledpin, LOW);
  delay(1000);
}
