Experiment-6

Aim of the Experiment:

Interfacing Air Quality Sensor (e.g. MQ135) ‚Äì Display data on LCD and switch on LED when pollution exceeds threshold

Hardware Required:

  • Arduino Uno ‚Äì 1
  • MQ135 Gas Sensor ‚Äì 1
  • 16x2 LCD Display ‚Äì 1
  • 10kŒ© Potentiometer ‚Äì 1
  • LED ‚Äì 1
  • 220Œ© Resistor ‚Äì 1
  • Breadboard ‚Äì 1
  • Jumper Wires ‚Äì As needed
  • USB Cable for Arduino ‚Äì 1

Software Required:

  • Arduino IDE
  • LiquidCrystal Library

Procedure:

  1. Connect the MQ135 sensor:
    - VCC to 5V, GND to GND
    - Analog OUT to A0
  2. Connect the LCD:
    - RS ‚Üí pin 12, EN ‚Üí pin 11
    - D4 ‚Üí pin 5, D5 ‚Üí pin 4, D6 ‚Üí pin 3, D7 ‚Üí pin 2
    - Potentiometer middle pin to V0 (contrast), sides to 5V and GND
  3. Connect the LED:
    - Positive leg (anode) of LED to pin 8 via 220Œ© resistor
    - Negative leg (cathode) to GND
  4. Upload the code:
    - Use the code below to read gas levels, display on LCD and light LED if threshold is crossed
  5. Observe Output:
    - LCD shows analog air quality value
    - LED turns ON if value is above specified threshold

Code:

#include <LiquidCrystal.h>

// LCD pin connections (RS, E, D4, D5, D6, D7)
LiquidCrystal lcd(12, 11, 5, 4, 3, 2);

const int mq135Pin = A0;  // MQ135 analog pin
const int ledPin = 8;     // LED connected to pin 8
int airQuality = 0;
int threshold = 400;      // Threshold value for pollution level

void setup() {
  lcd.begin(16, 2);
  pinMode(ledPin, OUTPUT);
  lcd.print("Air Quality Init");
  delay(2000);
  lcd.clear();
}

void loop() {
  airQuality = analogRead(mq135Pin);

  lcd.setCursor(0, 0);
  lcd.print("Air Quality:");
  lcd.setCursor(0, 1);
  lcd.print("Value: ");
  lcd.print(airQuality);
  lcd.print("   ");

  if (airQuality > threshold) {
    digitalWrite(ledPin, HIGH); // High pollution - turn ON LED
  } else {
    digitalWrite(ledPin, LOW);  // Normal - turn OFF LED
  }

  delay(2000);
}

Output

  • LCD displays air quality values.
  • LED turns ON when air quality value exceeds 400.
